# FreshMart Ghana: Answering Business Questions with SQL

> **Portfolio case study** · SQL · MySQL · Commercial analysis

## Business brief

Use transactional data to answer practical questions about sales, products and performance for a retail-style business scenario.

## SQL capabilities demonstrated

- filtering with `WHERE`
- sorting and ranking with `ORDER BY` and `LIMIT`
- aggregation with `SUM`, `COUNT` and `AVG`
- category analysis with `GROUP BY`
- group filtering with `HAVING`
- business rules with `CASE`
- combining related tables with joins
- top-N product and sales queries

## Analytical process

1. Reviewed the business question and identified the required fields.
2. Confirmed table structures and relationships.
3. Wrote a focused query for each question.
4. Checked aggregations and grouping logic.
5. Ordered the output to make the business result easy to interpret.
6. Added a plain-language explanation below each query.

## Challenge addressed

A technically correct query is not automatically a useful analysis. Each output was framed around a specific decision, with clear aliases, purposeful sorting and concise interpretation.

## Business outcome

The project demonstrated how SQL can convert transactional records into understandable outputs for product, sales and inventory discussions.
