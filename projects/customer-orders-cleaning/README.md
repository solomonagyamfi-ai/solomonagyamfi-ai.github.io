# Customer Orders: Data Quality and Preparation

> **Portfolio case study** · Excel · Power Query · Data cleaning

## Business brief

Prepare a customer-orders dataset for reliable reporting by resolving missing values, inconsistent field formats and unsuitable data types.

## Analytical process

1. Inspected column quality, completeness and consistency.
2. Identified null values and reviewed what each missing value represented.
3. Standardised text fields and corrected data types.
4. Checked for duplicates and invalid values.
5. Preserved an auditable cleaning sequence in Power Query.
6. Reviewed the final structure before analysis and visualisation.

## Key challenge

Missing values should not automatically be replaced with zero, because zero may carry a genuine business meaning. The correct treatment depends on the column, the reason the value is missing and how the field will be used.

## Business outcome

The project produced a cleaner, analysis-ready data structure and a repeatable transformation workflow. This improves confidence in downstream totals, averages, categories and dashboard results.
